Browse For Speed is a server browser for the popular online racing sim Live For Speed. It is able to query demo/s1/s2 servers, lets you maintain a favourites list and keeps track of your friends. It also has a basic (but powerful) admin interface to allow you to control your server via the insim interface. See the screenshots/FAQ for more features.

Browse For Speed 0.5 - Released!

Browse for Speed 0.5 - The main program files. This requires .NET 2.0 - see below for a direct download link.

Version 0.5 is out! The stability of 0.4, with some great added features:

  • Added translation system. This allows anyone to plonk an appropriate xml file in the lang directory and be able to have a translated Browse For Speed. Several languages are included with this release. If you would like to be a translation maintainer, please contact us.
  • Added admin interface. This is currently a basic system allowing you to send and receive messages to/from your server, as if you were in-game. Supports direct connections and connecting via the insim relay system.
  • Added ability to start arbitrary number of programs before LFS.
  • Ability to add to favourites by IP address
  • Many miscellaneous improvements (keyboard shortcuts, improved cfg section etc)

Upgrade instructions: (from any version) Simply unzip into your existing folder, overwriting any files.

Arne Claassen's Insim Library. This library is used by the admin interface. The LFSLib.dll distributed with Browse For Speed is modified, to get the complete source, download 0.11b from his site and our patch which against version 0.11b.

The library was developed with MonoDevelop, and requires .NET 1.1. The front-end code requires .NET 2.0. This is the first project we have ever written in c#.
