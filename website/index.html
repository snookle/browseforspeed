Browse For Speed is a server browser for the popular online racing sim Live For Speed. It is able to query demo/s1/s2 servers, lets you maintain a favourites list and keeps track of your friends. It also has an admin interface to allow you to control your server via Insim. See the screenshots/FAQ for more features.

Browse For Speed 0.7d!

Browse for Speed 0.7d - The main program files. This requires .NET 2.0 - see below for a direct download link.

Version 0.7d is out! Changelog from 0.6:

  • Send updated version - 0.7d
  • Fixed Master Server problems - 0.7c
  • Fix more encoding problems
  • Improve list drawing - faster, no flashing, huge friends lists are now very fast
  • Coloured server names
  • Supports all the characters found in LFS (provided you have appropriate fonts installed, see the FAQ for details)
  • Added quick refresh to only refresh currently displayed servers
  • Favourite list includes Version column
  • 0.7b fixes a rare issue of starting LFS in the wrong mode

Upgrade instructions: (from any version) Simply unzip into your existing folder, overwriting any files.

Arne Claassen's Insim Library. This library is used by the admin interface. The LFSLib.dll distributed with Browse For Speed is modified. To build it, download 0.11b from his site and our patch.

The library was developed with MonoDevelop, and requires .NET 1.1. The front-end code requires .NET 2.0. This is the first project we have ever written in c#.
