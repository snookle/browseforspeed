Browse For Speed is a server browser for the popular online racing sim Live For Speed. It is able to query demo/s1/s2 servers, lets you maintain a favourites list and keeps track of your friends. It also has an admin interface to allow you to control your server via Insim. See the screenshots/FAQ for more features.

Browse For Speed 0.7!

Browse for Speed 0.7 - The main program files. This requires .NET 2.0 - see below for a direct download link.

Version 0.7 is out! Changelog from 0.6:

  • Fix encoding problems
  • Speed up list drawing, especially friends
  • Coloured hostnames
  • Supports all the characters found in LFS (providing you have appropriate fonts installed)
  • Favourite list includes Version column

If you want to be able to see all characters (Japanese, Cryllic etc) properly, you need to install additional font support. Click Start->Control Panel->Regional and language Options->Languages and check the two checkboxes in "Supplemental language support". You will need your Windows install CD. You need to reboot for this support to work properly. Doing this will also enable you to see the characters properly on LFSWorld.
Note that not installing this won't affect functionality of BFS, you will simply not be able to see these characters (they'll show up as a square).

Upgrade instructions: (from any version) Simply unzip into your existing folder, overwriting any files.

Arne Claassen's Insim Library. This library is used by the admin interface. The LFSLib.dll distributed with Browse For Speed is modified. To build it, download 0.11b from his site and our patch.

The library was developed with MonoDevelop, and requires .NET 1.1. The front-end code requires .NET 2.0. This is the first project we have ever written in c#.
