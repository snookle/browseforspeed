Browse For Speed is a server browser for the popular online racing sim Live For Speed. It is able to query demo/s1/s2 servers, lets you maintain a favourites list and keeps track of your friends. See the screenshots/faq for more features.

Browse For Speed 0.4 - Released!

Browse for Speed 0.4 - The main program files. This requires .NET 2.0 - see below for a direct download link.

Version 0.4 is out! The most noticeable change in this version is the ability to query Demo/S1 servers and much improved filtering. Changelog from 0.3:

  • Added ability to query Demo/S1 servers. These servers can also be added to favourites.
  • Added filtering options. Public/Private/Empty/Full/Cars filtering works both server and client side, and ping/track filtering is done client side.
  • Fixed friends being deleted if the query fails initially (if lfsworld is down for example)
  • Fixed friends being deleted if the query fails initially (if lfsworld is down for example)
  • Fixed configuration updating

Upgrade instructions: (from any version) Simply unzip into your existing folder, overwriting any files.
